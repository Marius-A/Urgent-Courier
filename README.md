# Urgent Courier #

**Urgent Courier** is actually made up of two java applications (web and desktop).

For the *web* app I used **Spring MVC** , and for *desktop* app I used **Swing** (for interface) , **Itext**(to create invoices ) , and for interacting withh the database I used **Hibernate**.
